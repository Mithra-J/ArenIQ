# ⚙️ ArenIQ — Backend Server

The Node.js backend that powers ArenIQ's alert engine, complaint routing, and authority escalation logic.

---

## What It Does

- Reads the `encroachment_report.json` generated by `ndwi_detection.py`
- Maps each detected zone's GPS centroid to the responsible local authority
- Dispatches push alerts via **Ntfy.sh**
- Tracks authority response time — if no action within deadline, escalates to higher official
- Receives and stores citizen reports submitted from the Flutter app
- Serves the React dashboard with live encroachment data

---

## API Endpoints

| Method | Route | Description |
|---|---|---|
| GET | `/api/encroachments` | Fetch all detected encroachment zones |
| POST | `/api/reports` | Receive citizen report from Flutter app |
| POST | `/api/alert` | Trigger push alert to authority |
| PUT | `/api/encroachments/:id/resolve` | Mark encroachment as resolved |
